# Master-Habilidades

## 🎯 Objetivo
Generar skills nuevas siguiendo los estándares de Antigravity: YAML frontmatter, instrucciones claras y ejemplos de uso.

## 📋 Instrucciones Paso a Paso
1. Analizar si la tarea es repetitiva.
2. Identificar el tipo de skill (Automatización, Transformación, etc.).
3. Generar el Frontmatter YAML.
4. Escribir el cuerpo del SKILL.md.
5. Generar assets opcionales (scripts, examples).
