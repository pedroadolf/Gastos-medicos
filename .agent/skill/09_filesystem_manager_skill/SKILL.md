# 📁 Filesystem Manager

## Descripción
Acceso estandarizado al sistema de archivos local para lectura, escritura y navegación.

## 🛠️ Capacidades
- Lectura y escritura de archivos locales.
- Exploración de directorios.
- Validación de rutas.

## 🚀 Prompt de Uso
"Usa la herramienta `filesystem` para listar los archivos en la raíz del proyecto."
