# 🧠 Knowledge Manager

## 🎯 Objetivo
Consultar automáticamente Knowledge Items (KIs) para proporcionar información de dominio específico sin inventar datos.

## 🔧 Instrucciones de Uso
1. Detectar el tema solicitado.
2. Buscar en `./knowledge/` por prioridad.
3. Leer y citar la fuente.
4. Si no existe, sugerir creación.

## 🚨 Restricciones Críticas
1. NUNCA inventar datos.
2. SIEMPRE citar la fuente.
